# SkyreCraft

A modern-maintained recreation of the golden age of Minecraft modding.

SkyreCraft is a large Minecraft 1.7.10 modpack built around the classic magic, technology, exploration, automation, and building mods of the era.

## Project Status

SkyreCraft is currently in early development.

## Initial Goals

- Minecraft 1.7.10
- Forge 10.13.4.1614
- Extensive Thaumcraft 4 addon support
- Major technology and magic mods
- Expanded biomes, dimensions, dungeons, mobs, and structures
- Modern stability and performance fixes
- Prism Launcher distribution
- Dedicated multiplayer server package

## Development Milestones

- `0.1` — Foundation and stability
- `0.2` — Thaumcraft core
